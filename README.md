# langchain-examples

## 初始化虚拟环境

```shell
# 初始化虚拟环境
uv venv .venv
source .venv/bin/activate
uv pip install "langchain[all]"
# 更新对应api-key
mv .env.example .env
```

## 项目介绍

langchain是常见的ai开发框架，本项目是langchain的一些样例，主要是为了学习langchain的使用。

## 获取API-KEY

我们使用openrouter作为大模型的入口, 通过[链接](https://openrouter.ai/settings/keys)获取对应key。
